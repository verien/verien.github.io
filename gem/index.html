Rails - Gem (持續更新)

Dec 14, 2015

  1. gem “devise”
    • 一套使用者認證(Authentication)套件，是Rails社群中最廣為使用的一套。
  2. gem “bootstrap”
    • 熱門的網站開發用的前端 CSS 套件。 ‘bootstrap/affix’ ‘bootstrap/alert’ ‘bootstrap/button’ ‘bootstrap/carousel’ ‘bootstrap/collapse’ ‘bootstrap/dropdown’ ‘bootstrap/modal’ ‘bootstrap/popover’ ‘bootstrap/scrollspy’ ‘bootstrap/tab’ ‘bootstrap/tooltip’ ‘bootstrap/transition’
  3. gem “simple_form”
    • 讓表單寫起來更乾淨，也支援 bootstrap 樣式，也支援 devise。
  4. gem “carrierwave”
    • Carrierwave是Rails應用程式裡面上傳檔案的簡單工具，上傳的檔案種類不限。
  5. gem “mini_magick”
    • 針對圖片進行一些處理，尤其是像是等比例縮放這種事情更是難不倒它。
  6. gem “font-awesome-rails”
    • 字體
